# ועדות רפואיות — צ'קליסט מסמכים חכם (btl-guide-light)

כלי אינטראקטיבי לבדיקת מסמכים נדרשים לוועדות רפואיות בביטוח לאומי.

## טכנולוגיה
- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui

## הרצה מקומית
```sh
npm install
npm run dev
```

## בנייה
```sh
npm run build
```
