# ועדות רפואיות — קיצור תהליכי ועדות

**קהל:** רופאים, פקידים, מבוטחים | **מצב:** הגדרה ראשונית

## מטרה
שימוש ב-AI לקיצור זמן ועדות רפואיות — מיון חכם, מיצוי מידע מקדמי,
והפחתת עומס הוועדות הרגילות.

## מבנה התיקיות
```
ועדות-רפואיות/
├── docs/          ← הגדרת הפרויקט, הצעת פיילוט
├── data/          ← נתוני תפוקת ועדות (מסונן)
├── experiments/   ← ניסויים ראשוניים
├── prompts/       ← פרומפטים לסיוע ברופא
└── outputs/       ← פלטים מאושרים
```

## הרצה
אין קוד — שלב הגדרה ראשוני.

## שלב הבא
